# Units Converter 🧮📏

## Description
This C++ project was designed to convert temperature and distance measuring units.

## How it works
First, is displayed a menu with the options of kinds of units the user can choose (currently 2, temperature and distance).
After selecting one of them, you need to choose:
- The unit you want to convert
- Its value
- What unit convert to

Then, the program calculates the final result, displays it and it goes back to the main menu. You can repeat this as many times as you like, just type 0 to quit.
