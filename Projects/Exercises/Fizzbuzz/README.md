# Fizzbuzz challenge

A simple fizzbuzz program in C++

## What it does

- Prints numbers from 1 to 100
- Replaces multiples of 3 with "fizz"
- Replaces multiples of 5 with "buzz"
- Replaces multiples of both with "fizzbuzz"
- Breaks a line each 10 numbers

## What i practiced

- Loops
- Conditionals
- Functions
- Output formatting
